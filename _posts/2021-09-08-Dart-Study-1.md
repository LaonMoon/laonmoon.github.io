* 참고 : 플러터 생존코딩 초판

```dart
void main(){
    print("Hello World!);
}
```
다트 문법은 main() 함수가 진입점(entry point)이다.

### 기본 문법
#### 주석
작성하는 방법은 총 3가지이다.
- // : 한 줄 주석
- /*...*/ : 여러 줄 주석
- /// : 문서 주석

#### 문장
문장 `statement`는 명령 단위이다. 문장 끝은 세미콜론(;)으로 표시한다.

#### 변수
변수는 데이터(값)를 담는 상자이다. 변수 종류를 타입 `type` 또는 자료형이라고 한다.
